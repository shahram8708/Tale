Lesson 1

What is TALE

TALE is a small language that reads like friendly English. You can tell the computer exactly what to do using words you already know.

Story-like commands Beginner friendly Not done yet
Analogy: Think of TALE as giving clear steps to a helpful robot friend. If you say it, the robot does it.

You write simple lines. The computer follows them in order. Blocks that belong together are indented and end with the word end.

Try it

Hello, world

say "Hello, traveler!"
say "TALE reads like a story."

Lesson 10

Functions

Bundle steps into a reusable action you can call anytime.

function reuse code Not done yet
Analogy: A recipe card. Once written, you can cook the same dish many times.

Reuse

Cheer function

function cheer name times
  repeat times
    say "You can do it, " + name + "!"
  end
end

Lesson 11

Errors and understanding mistakes

Mistakes happen. TALE lets you catch them so your program does not crash.

try / catch Not done yet
Remember: When something breaks, read the message. It tells you what went wrong and where.

Handle

Catch a crash

try
  total is 10 / 0
  say total
catch problem
  say "Oops: " + text(problem)
finally
  say "We keep going anyway."
end
